# BigTime

    Qt-based Python TimeClock Solution for businesses.

## Why BigTime?

    Unlimited employees, simple UI, easy setup, cross-platform, server-sync, free and open-source.

## What features do we offer?

    * Clock in/out
    * Time-server syncing
    * Company database server syncing
    * Simple paystub generation for contractors/managers
    * Database backups

### Management features

    * Simple employee database management for managers
    * Time log viewing and editing

## How To Install?

### Are you a windows user?

    Simply download the exe to your company document folder, and run.

### Not on Windows?

    1. Install Python
    2. Run this command
       `python -m pip install pyinstaller`
    3. Open a new command terminal, then run this command
       `python -m pyinstaller TimeClock.spec`
    4. Create a shortcut to the dist file, and run!

#### If you know how to do so, please submit a pull request to the executable/package file included in it's respective operating system folder, to ease the installation of cross-platform users like yourself
